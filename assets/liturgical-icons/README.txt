SOURCE-RESOLUTION TRANSPARENT SVG SET

Each SVG contains a lossless RGBA PNG crop derived directly from the uploaded
generated sheet without resampling the artwork pixels.

No icon artwork was resized, downsampled, traced, recolored, sharpened, or
redrawn. The white crop background, printed routing number, and detached JPEG
compression fringe were removed. The transparent bounds were trimmed and four
transparent pixels of outer padding were retained for responsive placement.

The SVGs contain no names, numbers, or labels.

The supplied source is a JPEG raster image. Embedding the decoded pixels in a
lossless RGBA PNG preserves the generated artwork while allowing the PWA theme
to render it without a white rectangle. Editable vector paths would require
tracing and would not preserve the source artwork.
